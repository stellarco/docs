# API Keys

## List API keys

```
GET /api/v1/api_keys
```

Returns all keys for the workspace (keys are masked except the last 4 characters).

## Create an API key

```
POST /api/v1/api_keys
```

```json
{ "name": "CI Pipeline" }
```

Returns the full key **once**. Store it securely.

## Delete an API key

```
DELETE /api/v1/api_keys/:id
```

Immediately revokes the key. Returns `204 No Content`.
