# Tags

## List tags

```
GET /api/v1/tags
```

## Create a tag

```
POST /api/v1/tags
```

```json
{ "name": "ux", "color": "#4248CE" }
```

## Update a tag

```
PUT /api/v1/tags/:id
```

## Delete a tag

```
DELETE /api/v1/tags/:id
```
