# Create Feature

## Request

```
POST /api/v1/features
```

```json
{
  "title": "Add dark mode",
  "description": "Support system preference and manual toggle.",
  "status": "Planned",
  "assignee_id": "usr_abc123",
  "tags": ["frontend"],
  "properties": {
    "priority": "High"
  }
}
```

**Required:** `title`

## Response

`201 Created`

```json
{
  "data": {
    "id": "feat_001",
    "title": "Add dark mode",
    "status": "Planned",
    "assignee": {
      "id": "usr_abc123",
      "name": "Alice Chen"
    },
    "tags": ["frontend"],
    "properties": {
      "priority": "High"
    },
    "created_at": "2025-01-15T10:00:00Z"
  }
}
```
