# Categories

## List categories

```
GET /api/v1/categories
```

## Create a category

```
POST /api/v1/categories
```

```json
{ "name": "Feature", "color": "#D1F96C" }
```

## Update a category

```
PUT /api/v1/categories/:id
```

## Delete a category

```
DELETE /api/v1/categories/:id
```
