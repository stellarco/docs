# List Insights

## Request

```
GET /api/v1/insights
```

**Query parameters:**

| Param | Type | Description |
|-------|------|-------------|
| `page` | integer | Page number |
| `per_page` | integer | Results per page (max: 100) |
| `tag` | string | Filter by tag name |
| `source` | string | Filter by source |
