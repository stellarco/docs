# Insight Comments

## List comments

```
GET /api/v1/insights/:id/comments
```

## Create a comment

```
POST /api/v1/insights/:id/comments
```

```json
{
  "body": "This matches the dark mode request from last week."
}
```

Returns `201 Created`.
