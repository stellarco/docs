# Sign Up

Create a new account.

## Request

```
POST /api/v1/auth/signup
```

```json
{
  "name": "Alice Chen",
  "email": "alice@company.com",
  "password": "secure-password"
}
```

## Response

```json
{
  "data": {
    "token": "sk_live_abc123...",
    "user": {
      "id": "usr_abc123",
      "name": "Alice Chen",
      "email": "alice@company.com"
    }
  }
}
```

## Errors

| Code | Reason |
|------|--------|
| `422` | Email already taken or validation error |
