# Get Current User

Returns the authenticated user's profile.

## Request

```
GET /api/v1/auth/me
```

## Response

```json
{
  "data": {
    "id": "usr_abc123",
    "name": "Alice Chen",
    "email": "alice@company.com",
    "role": "admin",
    "avatar_url": "https://usestellar.com/avatars/usr_abc123.jpg"
  }
}
```
