# Team Roles

Stellar has three roles with different permission levels.

## Admin

Full access to everything:

- Manage billing and subscription
- Invite and remove members
- Change member roles
- All maker permissions

## Maker

The standard role for product team members:

- Create, edit, and delete features, insights, and announcements
- Create team views
- Manage tags, properties, and categories
- Publish and schedule announcements
- Configure portal settings

## Contributor

A lighter role for people who give input but don't manage the product backlog:

- Create insights
- Add comments on features and insights
- View all features and views
- Cannot create or edit features
- Cannot publish announcements
- Cannot change settings

## Choosing roles

- **Product managers, designers, engineers** → Maker
- **Support, sales, customer success** → Contributor
- **Team leads who handle billing** → Admin

You can change a member's role anytime under **Settings → Team**.
