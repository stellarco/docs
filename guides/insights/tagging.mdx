# Tagging Insights

Tags let you categorize insights by theme, product area, priority, or whatever taxonomy makes sense for your team.

## Creating tags

Go to **Settings → Tags** to create your tag vocabulary. Examples:

- `ux`, `performance`, `pricing`, `onboarding`
- `bug`, `feature-request`, `question`
- `enterprise`, `smb`, `consumer`

## Applying tags

Add tags when creating an insight, or batch-tag from the insights list view. Click any insight, then add tags in the sidebar.

## Tag-based filtering

Use tags in [Views](/guides/views/overview) to build filtered lists. For example: "All insights tagged `pricing` from the last 30 days."

## Auto-tagging

Stellar's AI can suggest tags based on insight content. Enable this in **Settings → AI → Auto-tag insights**.
