Transform your insights, features, and announcements with rich media embeds. Simply paste a link from supported platforms and watch it transform into an interactive embed.

[image]

## How Embeds Work

1. Copy a link from a supported platform
2. Paste it on its own line in the editor
3. The link automatically converts to an embed
4. Preview shows the embedded content

It's that simple - no special syntax required!

## Supported Platforms

### YouTube
Share video tutorials, demos, or presentations:
```
https://www.youtube.com/watch?v=dQw4w9WgXcQ
```

**Shows as:** Interactive YouTube player

[image]

### Loom
Perfect for quick screen recordings and walkthroughs:
```
https://www.loom.com/share/abc123def456
```

**Shows as:** Embedded Loom video with controls

[image]

### Wistia
Professional video hosting for product videos:
```
https://company.wistia.com/medias/abc123
```

**Shows as:** Wistia player with analytics

[image]

### Figma
Share designs and prototypes directly:
```
https://www.figma.com/file/abc123/Design-Name
```

**Shows as:** Interactive Figma embed

[image]

### X (Twitter)
Embed tweets for social proof or updates:
```
https://twitter.com/username/status/1234567890
```

**Shows as:** Full tweet with media and interactions

## Best Practices

### For Product Demos
- Use **Loom** for quick feature walkthroughs
- Embed **YouTube** for polished tutorials
- Keep videos under 5 minutes for engagement

### For Design Context
- Embed **Figma** files to show mockups
- Link to specific frames or prototypes
- Update embeds as designs evolve

### For Social Proof
- Embed customer **tweets** praising features
- Show community excitement with X posts
- Add context around embedded tweets

[image]

## Tips for Great Embeds

**One Per Line** - Place each embed URL on its own line

**Add Context** - Introduce embeds with text explanation

**Check Permissions** - Ensure embeds are publicly accessible

**Preview First** - Check how embeds look before publishing

## Troubleshooting

### Embed Not Showing?
- Ensure URL is on its own line
- Check if content is public
- Verify you're using a supported platform
- Try refreshing the editor
