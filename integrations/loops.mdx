## Why Loops?

- **Developer-friendly** - API-first design
- **Beautiful emails** - Modern templates
- **Smart automation** - Trigger-based workflows
- **Great analytics** - Track engagement

## Quick Setup

1. Navigate to **Portal** → **Settings** → **Email Marketing**
2. Select **Loops**
3. Configure:
   - **API Key** - From Loops settings
   - **List ID** - Target list for subscribers
   - **Tag** - Applied on signup (e.g., "changelog-subscriber")
4. Save configuration

[image]

## Getting Your API Key

1. Sign in to [Loops](https://loops.so)
2. Go to **Settings** → **API**
3. Create new API key with **Contacts** permission
4. Copy and save securely

<Info>
Loops automatically creates contacts with the changelog tag, perfect for targeted campaigns.
</Info>
